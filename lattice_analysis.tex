\section{The Lattice Structure}

All groups embed into $\Pi = \prod_{i=1}^{\infty} \Z/2^i\Z$. The containment lattice is:

\begin{center}
\begin{tabular}{c}
$\Pi = \prod_i \Z/2^i\Z$ \\[2pt]
$|$ \\[2pt]
$K + \bar{G}$ \\[2pt]
$\diagup \quad \diagdown$ \\[2pt]
$K \qquad\qquad \bar{G}$ \\[2pt]
$\diagdown \quad \diagup$ \\[2pt]
$G = \bigoplus_i \Z/2^i\Z$ \\[2pt]
$|$ \\[2pt]
$0$
\end{tabular}
\end{center}

\noindent\textbf{Notation:}
\begin{itemize}
\item $G = \bigoplus_{i=1}^{\infty} \Z/2^i\Z$ (direct sum of cyclic 2-groups)
\item $K = \Ext^1(\Q/\Z, G)$ (cotorsion hull)
\item $\bar{G} = \{(a_i) \in \Pi : \exists k,\, 2^k a_i = 0 \;\forall i\}$ (torsion-complete group)
\item $\Pi = \prod_{i=1}^{\infty} \Z/2^i\Z$ (full product)
\end{itemize}

\section{Key Structural Results}

\begin{enumerate}
\item $K \cap \bar{G} = G$ (incomparable in containment)
\item $(K+\bar{G})/G \cong K/G \oplus \bar{G}/G$ (direct sum since one is TF, other is torsion)
\item $K/G \hookrightarrow \Pi/\bar{G}$ is the maximal divisible subgroup
\item $\Pi/(K+\bar{G}) = (\Pi/\bar{G})/(K/G)$ is reduced torsion-free
\end{enumerate}
